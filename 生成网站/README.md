# 网站生成器项目

这是一个静态网站生成器项目，用于快速创建和部署网站内容。

## 功能特点

- 支持 Markdown 文件转换为 HTML
- 自定义主题和模板
- 响应式设计
- 自动生成导航菜单
- 支持代码高亮
- SEO 优化

## 文件结构说明
```
生成网站/
├── index.html              # 网站主页面
├── about.html             # 关于我们页面
├── contact.html           # 联系我们页面
├── static/               # 静态资源目录
│   ├── css/             # CSS样式文件
│   │   ├── style.css      # 主样式文件
│   │   ├── responsive.css # 响应式样式
│   │   ├── products.css   # 产品相关样式
│   │   └── honors.css     # 荣誉相关样式
│   ├── js/              # JavaScript文件
│   │   ├── main.js        # 主要脚本文件
│   │   └── products.js    # 产品相关脚本
│   └── images/          # 图片资源目录
├── .cursorrules         # 编辑器配置文件
└── README.md            # 项目说明文档
```
